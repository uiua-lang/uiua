Announcing Uiua 0.17.0

You can read this post with full editor features here.

2025-09-23

Uiua 0.17.0 is now available!

You can find the full changelog here.

You can download pre-built binaries here.

Uiua is a general purpose array-oriented programming language with a focus on tacit code.

This release is mostly focused on bug and crash fixes; there are no major new features.

That said, there are still some notable smaller changes and additions in this release:

New Subscript Formatting

Subscripts now format from a , rather than __ like they did previously.

coup,3 2 3 5 # Try formatting# [2 3 5]

This makes them typable with one keystroke instead of three, which is nice as more things in the language use subscripts.

Primes in Binding Names

Binding names can now contain Unicode prime glyphs. These format from 's at the end of the name.

Try formatting this example:

A' = 1 B'' = 2 C''' = 3

Array Pack Syntax

It is common to wrap the results of a ⊃ fork or ⊓ bracket function pack in an array to collect the results.

[⊃(⊢|⊣|/+)] [3 8 2 9 5] # [3 5 27] {⊓(⍉|+|⇡)} [1_2 3_4] 6 10_20 5 # [3 5 27] # ┌─ # 1 3│[16 26]│[0 1 2 3 4] # 2 4│ │ # ┘

The parentheses of the function pack can be replaced with the array brackets.

⊃[⊢|⊣|/+] [3 8 2 9 5] # [3 5 27] ⊓{⍉|+|⇡} [1_2 3_4] 6 10_20 5 # [3 5 27] # ┌─ # 1 3│[16 26]│[0 1 2 3 4] # 2 4│ │ # ┘

⨂ indexin

⊗ indexof is now deprecated. It was almost always used along with ˜ backward , and this change also makes its argument order convention match that of ∊ memberof .

It has been replaced with ⨂ indexin , which has the exact same functionality, except its arguments are flipped.

F ← ⨂"abcde" F "beefcake" # [1 4 4 5 2 0 5 4]

⊗ indexof will be automatically replaced in existing code.

Stabilized Primitives

  • ⧆ occurrences

    ⧆ "lego helmet" # [0 0 0 0 0 0 1 1 0 2 0] ⧆ "abbacaab" # [0 0 1 1 0 2 3 2] [0 0 0 0 0 0 1 1 0 2 0] ⧆₁ "abbacaab" # [1 1 0 0 1 0 0 0] [0 0 1 1 0 2 3 2] [0 0 0 0 0 0 1 1 0 2 0]

    • Replaces and deprecates ◰ unique

    • ◰ unique will be automatically replaced

  • ˙ self

    ˙× 5 # 25 ˙⊟ 10 # [10 10] 25 ˙⊟₃ τ # [τ τ τ] [10 10] 25

Notable Breaking Changes

Subscripted ⊂ join now joins that many arrays.

⊂₃ "wow" @ "cool" # "wow cool" ⊂₄ 1 2_3 4_5_6 7 # [1 2 3 4 5 6 7] "wow cool" /◇⊂₃ @, {"dog" "cat" "bird"} # "dog,cat,bird" [1 2 3 4 5 6 7] "wow cool"

⊙ dip and ⟜ on function packs no longer apply their modifier to the leftmost function. This makes certain patterns easier to read and write.

⊙(⊂|÷|+) 1 2 3 5 # [1 4] ⊂⊙(÷⊙+) 1 2 3 5 # [1 4] [1 4]

⍜ under △ shape now tiles the original array to match the new shape instead of ↯ reshape ing.

⍜⊙△⊂ 2_3 ["ab" "cd"] # ╭─ # ╷ "ab" "ab" "ab" # ╷ "cd" "cd" "cd" # ╷ # "ab" "ab" "ab" # "cd" "cd" "cd" # ╯

⇌ reverse , ⍉ transpose , and ↻ rotate no longer work through boxes.
